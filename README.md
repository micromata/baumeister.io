> This repository contains the sources of the [Baumeister website](https://baumeister.io).

## Quick install guide

	$ git clone https://github.com/micromata/baumeister.io.git
	$ cd baumeister.io
	$ npm install

## Run

	npm start

_Check`npm run` for additional scripts._

## Dependencies

- Node.js (>=6.0.0)
- Gulp

## Deployment
